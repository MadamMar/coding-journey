It is finally time to get to the exciting part of web design, implimenting CSS into HTML.

External CSS (the most commonly used method) is created in a separate file and is added within the 'head' tags of the HTML, it uses a self-closing 'link' tag. This method is good when a website has many pages that share the same style.

Internal CSS (or embedded CSS) is created within the HTML file itself within the 'head' tags, they use opening and closing 'style' tags. This method is useful a page that will have a different style (Note, depending on the amount of rules and declarations, this will make the HTML quite large).

Inline CSS is used to add styles directly to HMTL elements using the 'style="attribute: value;"'. This method is not recommended often because it could get a bit messy by bloating the HTML. You also run the risk of CSS overrifing it.
