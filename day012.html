An example using an arithmetic operation with two literal numbers:
let x = 100 + 50;
Or with variables:
let = a + b;
Or with expressions:
let = (100 + 50) * a;
As established, the Operators are the symbols that tell the numbers what to do, the numbers are called Operands. Example: 100( the perand) + (the operator) a (the operand)
On a side note in JavaScipt, in addition to 'addition', the + operator is also used as a concatenation when the operands are in quotations, these are called a string. Example:
let x = "10";
let y = "20";
let z = x + y;
The result would be z = 1020. This will be the result even if one of the numbers is surrounded by quotations.

JavaScript interprets numbers from left to right, so in this example, + is an addition and a concatenation:
let x = 10;
let y = 20;
let z = "30";
let result would = 3030. JavaScript would first read 10 + 20 to = 30, it would be 30 + "30" (a concatenation), resulting in 3030.
In Summary, let x = 100; is a number, let x = "100"; is a string. This is only important when using +. Numbers in quotations will work with every other operator normally. "100" / "10" will be 10.

Lastly, in order of operations, JavaScript also uses BEDMAS as learned in school. Brackets, Exponents, Division (or) Multiplication, Addition (or) Subtraction.

Types of Numbers

Integers
Floating-point numbers without a fraction, positive or negative
Floats (Floating Point Numbers
Have a decimal point or decimal places
Doubles
A specific type of Floating point number that have greater precision than standing FPN, they are accurate to a greater number of decimal places

Types of Number Systems

Binary
The lowest level language of computers, all 0s and 1s
Octal
Base 8, uses 0-7 in each column
Hexadecimal
Base 16, uses 0-9 and then a-f in each column (like the numbers for colors in CSS)

Helpful Operators/Methods

typeof will show what datatype a variable contains. Exampe:
console.log(typeof 42);
console.log(typeof '42')
>"number"
>"string"

toFixed() method will round a number to a fixed number of decimal places. Example:
const manyDecimals = 5.3234546;
manyDecimals;
const twoDecimalsOnly = manyDecimals.toFixed(2);
twoDecimalsOnly;

Number() can be used to change something like a string (a number within quotations) into a number. Example:
let myNumber="5";
myNumber = Number(myNumber) + 5;
the result will be >10, not 55.
