The Two Axes of Flexbox

Axis One, The Main Axis
Is defined by the flex-direction, of which the four possible vaules are

  • row
  • row-reverese
  • column
  • column-reverse
    •
Row and Row Reverse will set the main axis along a vertical row from left to right, or right to left, in the inline direction.
Column and Column Reverse will set the main axis along a horizontal row from top to bottom, or bottom to top, in the block direction.

Axis Two, The Cross Axis
Is determined by the main axis, it is the columns/rows that run left-right or top-bottom (and reverse).

Start and End Lines

The start and end lines refer to where the box begins and ends. If using the row-reverse, the start would be on right hand side of the page and end on the left, reverse of what row would be.

Flex Container

As we've worked with already, Flex Containers are used to contain their content by setting a value using the container's display property to flex or inline-flex.
When that is set, the contents (or children) of the container become flex items. They will behave in certain ways

  • The items will display in a row if flex-direction is set to/defaulted to row
  • The items will start from the beginning of the main axis
  • The items will not stretch to fill the size of the cross axis
  • The flex-basis property is set to auto
  • The flex-wrap property is set to nowrap

Multi-line Flex Containers with Flex-Wrap

To cause items to wrap, use the flex-wrap: wrap; property and value. This way, if the items are too large to display in one line, they will wrap onto another one. With a value of nowrap, they will instead shrink to fit the container (this can also cause an overflow if the items can't fit/shrink small enough)

Flex-flow Shorthand and Values for Flex Propters

In order to control flex items, target them directly by using three properties: flex-grow, flex-shrink, and flex-basis. We already went through these in Day 8, Growing and Shrinking.
Predefined Shorthand Values:
flex: initial; resets the item to the initial values of Flexbox, items will not grow but they will shrink if necessary. It is the same as flex: 0 1 auto;
flex: auto; works similarly to initial but will also allow items to grow to fit its container. It is the same as flex: 1 1 auto;
flex: none; causes items to not change at all. It is the same as flex: 0 0 auto;
flex: [positive-numer]; often appears as flex: 1; or flex: 2; and so on ...This will set both grow and shrink to the same value with the flex-basis being 0. It is the same as flex: 1 1 0; or flex: 2 2 0;

Alignment, Jusitification and Distribution (of Free Space between Items)

Flexbox has the ability to align and justify items on the main-axis and cross-axis using the align-items property. This must be done within the container, not the items.

align-items:

  • stretch; aligns(or stretches) items to the height of the tallest container
  • flex-start; align items to the start of the flex container
  • flex-end; aligns items to the end of the flex container
  • center; align items to the center of the flex container
We can also use the justify-content property to align items on the main-axis

justify-content:

  • flex-start; flex-end; center; to line up at the start, end or center of the container
  • space-around; to space the items apart equally from each other and from the edges of the container
  • space-between; to space the items apart to fill up the container with the first item at the edge of the start of the container and the last item at the edge of the end
  • space-evenly; to space items evenly on either end to their full size
