Increment and Decrement Operators

To easily add a numeric value by just 1, we use the increment operator (++). To subtract by just 1, we use the decrement operator (--). Example:
let num1 = 4;
>4
num1++;
>5

Assigned Operators

The basics are simple and what we've already been through (let x = 3; sets x value to 3, let y = 4; sets y value to 4. x = y would now make x value 4, etc)
There are also options that are a bit more complex for assigning values. These are some of the more common ones:

Operator
Assignment
Purpose
Example
Shortcut for
+=
Addition
Adds the right value to the left value, creating a new variable value
x += 5;
x = x + 5;
-=
Subtraction
Subtracts the right value from the left value, creating a new variable value
x -= 5;
x = x - 5;
*=
Multiplication
Multiples the left value by the right value, creating a new variable value
x *= 5;
x = x * 5;
/=
Division
Divides the left value by the right value, creating a new variable value
x /= 5;
x = x / 5;

Example:
let x = 3;
let y = 4;
x *= y;
X would then become 12 (x = x * y; or x = 3 * 4;)

Sizing a Canvas Box

We'll be using Canvas API but focusing on the math portion of how it works. To make a box, you'll use the variable x (for width) and y (for height), these values are in pixels. Let's make a box of 50px height and width.
Example code for a Canvas box:

let x = 50;
let y = 50;
x = 50;
y = 50;
ctx.fillStyle = 'green';
ctx.fillRect(10, 10, x, y);

These values will return true/false when entered into the console. These 'booleans' can be used for numerous things, such as:

  • displaying whether a feature is on or off on it's text label
  • displaying a victory and game over message when a game is complete
  • displaying a proper seasonal message depending on what holiday it is
  • displaying the proper amount of zoo on a map depending on what is selected
