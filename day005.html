The Box Model

Everything on a page is a box/rectange, as displayed by this grey border around this sentence. That is this 'p's border on the site.

Even links and have boxes.
Everything fits together like a puzzle in an HTML's layout.

The elements sizes can be manipulated using padding, margin, and border.

  • Padding: Increases space between edge of a box and the contents inside
  • Margin: Increases space between the box and anything that is sitting next to it
  • Border: Adds space between the padding and margin

For this box, the text is the content. We'll make the border black, the padding light yellow. The margin can only be the invisible space that surrounds the box.

There are two types of boxes in CSS: block boxes and inline boxes. They can be added to the CSS using "display: border-box;"" or "display: inline-box". These boxes are considered Outer Display types

Block boxes will cause the box to:
(Some HTML elements that use block by default are h1 and p

  • Break onto a new line
  • Extend in the inline direction to fill the spaces available in its container
  • Follow the width and height properties
  • Push other elements away based on the padding, margin, and border

Inline boxes will cause the box to:
(Some HTML elements that use inline by default are a, span, em, and strong)

  • Doesn't break onto a new line
  • Does not follow width and height properties
  • Vertically it will not cause other inline boxes to move away from the box (padding, margins and border still apply)
  • Horizontally it will cause other inline boxes to move away from the box

Now for Inner Display types. These control what happens on the inside of the box, to the elements/contents. By default, elements are laid out in normal flow.
A common Inner Display used is flex and grid. These are also added using 'display:' attribute.
