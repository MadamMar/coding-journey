Block vs Inline

Block (display: block) appears on a page stacked on top of each other, each starting a new line, like lists that appear in separate lines independently.

Inline-Block (display:inline-block) mixes characteristics of both. They are like inline elements but use block like padding and margin. In most cases that need to line up boxes, Flexbox is generally a better option.

Divs and Spans

Div elements are block-level and are used like a container to group other elements together. They are useful to block different sections of a page and style them.

Span elements are inline-level and is used to group text content and inline HTML elements for styling. It is useful when styling within one element and no other semantic HTML element is appropriate.
