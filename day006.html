Borders, in Depth

Similar to Margins, you can set specific values for the borders using:

  • Individual Values -
  • border-top
    border-right
    border-bottom
    border-left
  • Values for all sides -
  • border-width
    border-style
    border-color

Or more specific combinations:

  • Border Width -
  • border-top-width
    border-right-width
    border-bottom-width
    border-left-width
  • Border Style -
  • border-top-style
    border-right-style
    border-bottom-style
    border-left-style
  • Border Color -
  • border-top-color
    border-right-color
    border-bottom-color
    border-left-color
