Growing and Shrinking

We can control the sizes using the flex: declaration (in the above examples, we used flex: 1). This declaration is shorthand for flex-grow, flex-shrink, flex-basis, in that order. When only one value is provided, it is the first, flex-grow.
Example) flex: 0 1 0%; would mean flex-grow: 0, flex-shrink: 1, and flex-basis: 0%.

flex-grow expects a single number value. When one is provided, all flex-item's growth factor will be the same size. You can also add unique values to specific boxes.
An example for changing a unique box is adding it to the css ...
Ex) .flex-container .two { flex: 2 1 0%; This would make the second Item a different size from it's parent value of 1.

flex-shrink is just like the flex-grow, but instead has a shrink factor. This value defauls to 1. To keep an item from shrinking to fit a container, set the value to 0.

flex-basis will set the initial size of a flex item, it is the baseline size that grow and shrink work off of. It defaults to auto if no value is given.

Axes

In flexbox, you'll use axes to turn your rows into columns.
There are two axes, the main axis and the cross axis. These are changed using the flex-direction:
flex-direction: row; makes the main axis horizontal (left to right).
flex-direction: column; makes the main axis vertical (top to bottom). A minor note when working with columns, the flex: basis will refer to height, not width.
