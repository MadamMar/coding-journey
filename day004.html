Opening DevTools

Inspect: Right click an element and select 'inspect'
View logged messages or run JavaScript: Press Ctrl+Shift+J
Open the last panel that was opened: Press Ctrl+Shift+I
From the Main Menu: Top-right: Select Settings > More Tools > Developer Tools

Adding a CSS Declaration to an Element
You can channge an element in the Styles tab to see the changes in realtime.
Adding a CSS Class to an Element
You can add a class b clicking .cls (next to the filter in the styles tab) to bring up a text box where you can add classes to the selected element. All classes that are applied to the selected element can be toggled on and off using the checkboxes.
Adding a Pseudostate to a Class
You can permanently apply a CSS pseudostate to an element using :hov (also next to the filter in the styles tab).
Changing the Dimensions of an Element
You can use the Box Model (located at the bottom), which is an interactive diagram in the styles tab that can change the width, height, padding, margin and/or border length of an element. Simply double-click any value you want to change and add a value, it defaults to pixels/px (ex, 100, 25%, or 10vw).
